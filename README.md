## What is Sevanta?

**Sevanta** is a smart desk management platform that enables users to control, monitor, and manage LINAK height-adjustable desks through a modern web interface. The system integrates embedded hardware (Raspberry Pi Pico W) with a full-stack web application and provides real-time desk control, scheduling, and analytics.

<details>
<summary>Features [View More]</summary>

### Desk Management

- Real-time height adjustment and control
- Desk reservations and scheduling
- Automated height adjustments based on schedules
- Desk status monitoring and reporting

### User Features

- User authentication and management
- Personalized desk height preferences
- Posture cycle settings and reminders

### Analytics & Reporting

- Usage analytics and tracking
- Desk usage patterns and height change logs
- Issue reporting

### Integration

- MQTT-based real-time communication
- WiFi2BLE simulator for testing
- RESTful API for external integrations
- Docker containerization support
